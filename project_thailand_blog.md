A static HTML/CSS/JS blog website for the team's 2-week MBA field project in Thailand with Hilltribe Organics.

**Why:** Final deliverable for the MBA course — a daily blog instead of a Word doc.

**How to apply:** When editing this project, all content lives in `js/data.js`. Site has two pages: `index.html` (landing) and `blog.html` (interactive map + daily blog). Deploy target is GitHub Pages.
